Katana Release Notes

August 31, 2018 - 2:10 PM
NEW Navigation All issue items (the column of cards on the left) now have clickable links that will take you to the individual results. You must first select the card, then click on the link to pull the issue into view
UPDATE Once there are no "untouched" issue cards (i.e. the card is no longer white but is "approved", "skipped", "fixed" or "error[ed]"), the next issue item will automatically be selected, thus making navigation easier and requiring less clicking
UPDATE The heading of the second column on the results page has been changed from "Issues" to "Results"
August 22, 2018 - 1:25 PM
UPDATE An "Add All Results" button has been added to the Course Selection screen which will add all of the courses in the results table
UPDATE New tools now have "(BETA)" attached to their titles, indicating that they may be buggy and/or may not give you all of the results in the course. The user should be aware of this as it is given to you for testing and for feedback
FIX HTML General Editor - On occasion the HTML wouldn't show up in the current or the updatcode editor tabs. This has since been fixed
FIX BYUI Style Classes - On occasion the inserted style class would be all uppercase letterhas since been fixed
NEW TOOL Course Search (BETA) - Allows the user to search the course for any content. Most users should use the "Text" Search Type, unless otherwise trained in CSS Selectors or Regular Expressions
NEW TOOL Link Editor (BETA) - Allows the user to search the course for any specific link inside of iframe, img, and a tags, and replace all of the results with a new specified link
NEW TOOL Transcripts (BETA) - Allows the user to search the course for existing transcripts or potential missing transcripts. This does not find all missing transcripts, but does find many that may potentially be missing
NEW TOOL Item Locator (BETA) - Allows the user to search the course for assignments, quizzes, and discussions and will report to the user where the item is found (i.e. If a discussion is found under Assignments and Modules as well, or not)
August 9, 2018 - 2:24 PM
FIX Layering of various pages and popups have been patched and simplified
FIX Fixed bugs which prevented Katana from successfully compiling a production version
FIX BYUI Style Classes - Occassionally added an extra div when the style classes were found, but formatted incorrectly. These issues have been corrected
UPDATE Clicking on the background of the course selection page closes the page
UPDATE Scrollbars have been added in appropriate places
UPDATE BYUI Style Classes - The UPDATED HTML code editor tab is now editable in case the incorrect style classes are inserted, or the user desires to make additional changes
August 7, 2018 - 3:25 PM
FIX The button to expand and collapse the code editor no longer requires two clicks to work the first time, but now functions with one click.
FIX The HTML Editor would input text incorrectly on occasion, and would backspace incorrectly. This has been resolved.
FIX BYUI Style Classes - Occassionally showed the Current HTML incorrectly. These consistency issues have been resolved.
July 27, 2018 - 3:21 PM
NEW TOOL HTML General Editor - Allows the user to edit HTML anywhere in the course. Can search for specific tags or phrases, or just retrieve all HTML from the course.
NEW TOOL Code Blocks - This tool identifies code blocks in course HTML that do not have the language added as a class.
NEW TOOL Replace CSS Classes - Allows the user to find elements with specific CSS classes and replace them with new ones.
NEW FEATURE A settings menu has been added.
NEW FEATURE The user can now select from three different themes (WIP). They are under the new settings menu.
NEW FEATURE The "Searching for issues..." message can now be switched to randomly generated phrases. They can be turned off in the settings menu.
UPDATE The course sidebar has been removed, and replaced with a new "tab" style bar, right underneath the navbar.
UPDATE The "Add Courses" button has been moved to the right of the new course tabs bar, and has been changed to "Edit Courses."
UPDATE The Issue Item list and the Issue List now have a scroll bar.
FIX Courses with large amounts of items discovered will now load much, much quicker.
REMOVED The pills showing off issue counts (approved, skipped, etc.) on the tool pages have been removed.
